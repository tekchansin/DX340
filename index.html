Seamless Installation

Automated setup in under 10 minutes. No hassle, maximum results.

1

Enable Developer Options

Go to Settings > About phone > Tap Build number 7 times. Then enable USB Debugging in System Settings.

2

Run Automated Script

Connect DX340 to PC and run the installer for your OS.

# For Windows

Double-click Windows-Anantar-Ultimate-Installer.bat

# For macOS / Linux

sh MacOS-Anantar-Ultimate-Installer.command

3

Magisk Setup

The script will open Magisk. If prompted for "Additional Setup", tap OK. The device will reboot and wait for 15 seconds.
