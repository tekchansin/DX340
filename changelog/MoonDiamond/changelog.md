## Introducing Deep Sleep mode.
- This mode will trigger when no music playing, no usb-dac, screen off in 15 mins
- This mode will put DX340 to powersaving ,disable wifi/bt and USB.
- That can reduce battery consumption to be 0.2-0.3 Watts.
- This mode will restore original state INSTANTLY when you open screen.
